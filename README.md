Projeto FiapRide - POO
Repositório de estudos para a disciplina de Programação Orientada a Objetos (FIAP - 3º Semestre).

Sobre o Projeto
O sistema simula o gerenciamento de passageiros e o monitoramento de baleias utilizando Java.

Funcionalidades
Classe Passageiro
Cadastro de nome, CPF e saldo.

Métodos para adicionar saldo e realizar pagamento de viagens.

Classe Baleia
Criação Flexível: Permite instanciar baleias com tamanhos diferentes via construtor.

Crescimento: Método que calcula o aumento do tamanho (0.5m por mês).

Migração: Lógica que valida se a distância percorrida é suficiente para uma migração.
